Material design buttons for Svelte

Кнопки в стиле материального дизайна для «исчезающего» фреймворка Svelte.

Дизайн кнопок взят с https://materializecss.com/buttons.html.

Код на GitHub.

Просто кнопка (Raised)

<Button waves></Button>
<Button waves icon="cloud" iconAlign="left"></Button>
<Button waves icon="cloud" iconAlign="right"></Button>

Cсылки в виде просто кнопки

<Button floating waves size="large" color="red" icon="add"></Button>

Плоская кнопка (Flat)

Обычно используется для уменьшения визуального шума внутри карточки или модального окна.

Flat buttons are used to reduce excessive layering. For example, flat buttons are usually used for actions within a card or modal so there aren't too many overlapping shadows.

<Button waves flat></Button>

Кнопка отправки (Submit Button)

Для отправки формы используйте тег <button type="submit"> вместо <input type="submit">.

When you use a button to submit a form, instead of using a input tag, use a button tag with a type submit.

<Button type="submit" waves icon="send" iconAlign="right" name="action"> Submit </Button>

Большая кнопка (Large)

Большая кнопка привлекает внимание :)

This button has a larger height for buttons that need more attention.

<Button waves size="large"></Button>
<Button waves size="large" icon="cloud" iconAlign="left"></Button>
<Button waves size="large" icon="cloud" iconAlign="right"></Button>

Маленькая кнопка (Small)

Маленькие кнопки удобнее для более плотной компоновки элементов интерфейса.

When mouse and keyboard are the primary input methods, this smaller button is useful for denser UI layouts.

<Button waves size="small"></Button>
<Button waves size="small" icon="cloud" iconAlign="left"></Button>
<Button waves size="small" icon="cloud" iconAlign="right"></Button>

Выключенная кнопка (Disabled)

Этот стиль можно применить ко всем типам кнопок.

This style can be applied to all button types.

<Button disabled size="large"></Button>
<Button disabled></Button>
<Button flat disabled></Button>
<Button floating disabled icon="add"></Button>
